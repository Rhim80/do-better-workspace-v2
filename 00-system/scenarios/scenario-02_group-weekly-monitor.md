## 미션 1: 그룹 4개 사업부 W21 한 장 요약

의장님이 월요일 아침에 5분 안에 흐름 잡을 수 있는 한 장.

> **(속으로)**: 의장님은 잡지 보듯이 한 장 훑고 끝낸다. 표를 들이밀면 안 본다. 본질 한 줄 + 사업부 4개 한 줄씩 + 다음 주 짚어볼 곳 한 줄.

**분석 요청**:
```
@sample-data/grandeclip-chairman/group_weekly_kpi.csv
@sample-data/grandeclip-chairman/amcr_sales.csv
@sample-data/grandeclip-chairman/newmix_store_sales.csv
@sample-data/grandeclip-chairman/stayfolio_occupancy.csv
@sample-data/grandeclip-chairman/magazine_c_issues.csv
@templates/grandeclip-chairman/group-monitor-weekly.md 양식으로
W21 (2026-05-18~24) 그룹 통합 모니터 1페이지 만들어줘.

- 표지 한 줄 헤드라인은 이번 주 그룹 본질 한 문장
- KPI 4-up: 그룹 합산 / AMCR 비중 / 뉴믹스 평균 일매출 / 스테이폴리오 점유율
- 사업부 4개 한 줄 카드
- W14~W21 그룹 매출 트렌드 라인
- 이상 신호 최대 3건
- 의장 한 마디 자리는 비워두기
```

**결과 확인 후 추가 질문 예시**:
```
"의장님이 '다음 주 어디 들여다봐야 해?' 물으시면 어디라고 답해야 해? 데이터 근거로 1~2곳만 짚어줘"
```

## 미션 2: 의장 한 마디 자리 — 한 문장 제안

미션 1 끝낸 게 10시 5분. 보고서가 깔끔하게 떨어졌다. 그런데 의장 한 마디 자리가 비어 있다. 평소 같으면 비워두고 의장님이 미팅에서 직접 채우시는데, 이번 주는 시즌 피크 진입이라 미리 한 줄 제안해두면 흐름이 빠를 것 같다.

> **(속으로)**: 의장님은 디자이너 출신이라 "본질이 잡히는 한 문장"을 좋아하신다. 숫자가 아니라 흐름의 의미. 잡지 헤드라인 같은 톤.

**분석 요청**:
```
@sample-data/grandeclip-chairman/group_weekly_kpi.csv
W21 데이터 보고, 의장님 코멘트 자리에 들어갈 한 문장 3개 후보 제안해줘.
- 첫째: 숫자 중심 (어떤 숫자가 본질인지)
- 둘째: 흐름 중심 (어디로 가고 있는지)
- 셋째: 잡지 헤드라인 톤 (의장님 디자이너 감각에 맞게 — 짧고 함축적)

내가 그 중 하나 골라서 보고서에 넣을게.
```

**결과 확인 후 추가 질문 예시**:
```
"잡지 헤드라인 톤 후보가 좋은데, 한 단어만 더 다듬어줘. '시즌 피크 진입, 그 다음을 본다' 같이 다음을 암시하는 톤으로"
```
