# 그룹 통합 모니터링 양식 (의장 직속)

> 그란데클립코리아 산하 4개 사업부(어메이징크리·뉴믹스커피·매거진C·스테이폴리오) 주간 흐름을 한 화면에 요약. 의장이 월요일 오전에 5분 안에 훑을 수 있도록 1페이지 안에 마무리.

## 사용 데이터

- `@sample-data/grandeclip-chairman/group_weekly_kpi.csv` — 주차별 사업부 매출·점유율 (필수, 핵심 데이터)
- `@sample-data/grandeclip-chairman/amcr_sales.csv` — 어메이징크리 채널 분해용 (선택, 그룹 70% 차지하므로 분해해서 보여줄 가치)
- `@sample-data/grandeclip-chairman/newmix_store_sales.csv` — 뉴믹스 매장 분해 (선택)
- `@sample-data/grandeclip-chairman/stayfolio_occupancy.csv` — 스테이폴리오 국가별 (선택)
- `@sample-data/grandeclip-chairman/magazine_c_issues.csv` — 매거진C 호별 (선택)

## 출력 구조 (1페이지 안)

### 1. 표지 헤더 (Editorial Slab)
- 라벨: `GRANDE CLIP / GROUP WEEKLY MONITOR`
- 헤드라인: 한 줄. 이번 주 그룹 한 줄 요약. ("어메이징크리 시즌 피크 진입, 뉴믹스 가정의 달 효과 잔존, 매거진C Vol.14 안정")
- 메타: 주차 (W21) + 기간 + 작성자.

### 2. 그룹 통합 KPI — 4-up 카드
- 그룹 합산 매출 (전주 대비 ▲/▼)
- 어메이징크리 매출 + 그룹 내 비중 (%)
- 뉴믹스 매장 평균 일매출
- 스테이폴리오 평균 점유율
- 변동 색상: 부정 GC Red, 긍정 잉크색 ▲.

### 3. 사업부별 한 줄 카드 (4개)
- 각 사업부 카드 좌측에 `{typography.label-uppercase}` 라벨 (AMCR / NEWMIX / MAGAZINE C / STAYFOLIO).
- 카드 1줄: 매출 + 변동 + 한 문장 코멘트 ("시즌 피크 진입 / 베스트 SKU 재고 임계").
- 코멘트는 의장이 5초 안에 읽고 흐름 잡을 수 있도록 짧게.

### 4. 주간 트렌드 라인 (8주)
- W14~W21 그룹 합산 매출 라인 차트.
- 무채색 라인 + 이번 주만 GC Red 점.
- 차트 아래 1문장 — 가장 큰 변동 주차에 대한 1문장 해석.

### 5. 이상 신호 / 주목할 점 (있을 때만, 최대 3건)
- 부정 신호: 채널 매출 -20% 이상, 재고 임계, 점유율 급락.
- 긍정 신호: 신규 채널 매출 진입, 시즌 SKU 매진.
- 모두 1줄씩. 출처 표기 ("AMCR amcr_sales W21 자사몰").

### 6. 의장 한 마디 자리 (Section Quote)
- 비어 있는 채로 두어도 OK. 의장이 직접 채우거나, 보고하는 사람이 미리 한 문장 제안.

## 검출 규칙

- **사업부 매출 급변**: 전주 대비 ±15% 이상 → 카드에 ▲/▼ 강조.
- **그룹 비중 변화**: 어메이징크리 비중이 80% 초과 또는 60% 미만 → 코멘트 추가.
- **점유율 임계**: 스테이폴리오 평균 점유율 < 0.50 → 알림.
- **매거진C 발행 주기**: 분기 발행 외 주차에 매출 잡히면 캐리오버 매출로 코멘트.

## 길이 제약

- **무조건 1페이지**. A4 1장 또는 데스크탑 1뷰.
- 카드는 4-up, 차트는 1개, 이상 신호 최대 3건.
- 페이지 길어지면 액션 아이템·디테일은 별도 보고서로 분리.

## 톤 가이드

- 결론·요약 위주. 디테일은 사업부별 상세 보고서로 위임.
- 의장 시점에서 "다음 주 어떤 사업부를 들여다봐야 하는가"가 한 눈에 보여야 함.
- 자기 PR·과한 수식어 금지.
- 추측은 "추정" 표기.
