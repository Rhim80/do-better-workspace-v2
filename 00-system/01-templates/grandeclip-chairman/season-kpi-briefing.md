# 어메이징크리 시즌 KPI 브리핑 양식

> 어메이징크리(AMCR) 매장·채널·SKU 단위 시즌 성과를 임원이 한 화면에 읽을 수 있도록 정리하는 양식. 의장 보고용·내부 임원 미팅 둘 다 사용. 1~2페이지 안에서 마무리.

## 사용 데이터

- `@sample-data/grandeclip-chairman/amcr_sales.csv` — 채널·SKU·시즌별 매출 라인 (필수)
- `@sample-data/grandeclip-chairman/amcr_product_master.csv` — SKU 정보·원가·마진 (필수)
- `@sample-data/grandeclip-chairman/amcr_inventory.csv` — 시즌 재고·재고일수 (선택, 재고 이슈 다룰 때)

## 출력 구조

### 1. 표지 — Editorial Slab Header
- 상단 풀블리드 잉크 영역. 흰 텍스트.
- 라벨: `AMAZINGCRE SS25 / WEEKLY KPI BRIEFING`
- 헤드라인: 한 줄. 이번 주 본질 한 문장. ("매출은 시즌 곡선 따라 올라왔으나 슬림팬츠 화이트가 시즌 끝까지 못 갈 위험")
- 메타: 주차 + 데이터 기간 + 작성자.

### 2. 핵심 KPI — 4-up 카드
- 이번 주 매출 (전주 대비 ▲/▼ %)
- 이번 주 주문 건수 (전주 대비 ▲/▼ %)
- 채널 1위 + 비중 (자사몰 / 백화점 등)
- 베스트 SKU 1위 + 점유율
- 변동 색상: 부정(▼)은 GC Red, 긍정(▲)은 잉크색 (▲ 표식만으로 표현).

### 3. 채널별 매출 — 표 + 짧은 인사이트
- 표: 채널명 / 매출 / 비중 / 전주 대비 / 평균 객단가
- 강조 행: GC Red Soft 배경 — 가장 큰 변동 채널 1행.
- 표 아래 1문장 인사이트: "왜 그 채널이 움직였는가" (날씨·시즌·이벤트 등 추정).

### 4. SKU TOP 5 — 표
- SKU / 상품명 / 카테고리 / 판매 수량 / 매출 / 마진율
- 마진율 표기 — 의장이 매출 다음으로 보는 숫자.

### 5. 이상 신호 (있을 때만)
- **재고 임계** (`weeks_of_stock` < 1.5): 베스트 SKU에 보충 필요한 품목. 표 1줄씩.
- **할인율 급증** (특정 채널 평균 할인율 > 0.20): 채널 1줄씩.
- **시즌 SKU 부진** (SS25인데 4주 누적 판매 미달): 카테고리 단위로 1줄.
- 이상 신호 없으면 섹션 자체를 생략 (잡지 톤 — 채우려 들지 말 것).

### 6. 의장 코멘트 자리 (Section Quote)
- 좌측 GC Red 4px 마커 + IBM Plex Serif Italic 인용구 톤.
- 비워두면 보고하는 사람이 한 줄 채움 ("이번 주 흐름을 한 문장으로 말씀드리면..." 형식).

### 7. 다음 주 액션 — 3줄 이내
- 백캐스팅 톤: "다음 주 끝났을 때 무엇이 달라져야 하는가" 1~3개.
- 액션마다 담당자 (이름 없으면 "MD팀" 같은 팀 단위).

## 검출 규칙 (Anomaly Thresholds)

- **재고 임계**: `weeks_of_stock < 1.5` → "긴급보충".
- **재고 과잉**: `weeks_of_stock > 12` → "과재고 — 시즌 종료 전 처리 검토".
- **채널 매출 급변**: 전주 대비 ±20% 이상 → 채널 행 강조.
- **할인율 과다**: 채널 평균 할인율 > 0.20 → "마진 누수 의심".
- **카테고리 부진**: 카테고리 매출 점유율이 전주 대비 -5%p 이상 → "카테고리 흐름 변화".

## 길이 제약

- A4 1~2페이지. 데이터 더 들어가면 페이지 추가, 표 압축 금지.
- 본문은 잡지 본문 톤 — 한 문단 3~5줄 이내.
- 차트는 1~2개. 더 필요하면 별첨으로 분리.
- 표는 한 페이지에 최대 2개. 더 필요하면 페이지 분할.

## 톤 가이드

- 임원 보고 톤. 결론·근거 순서.
- "그래서 무엇을 해야 하는가" 를 마지막에 반드시 1~3줄.
- 추측·의견에는 "추정" / "가설" 표기.
- 의장 거명 회피 — "의장님이 보실 때 ..." 같은 화법 사용 안 함. 그냥 사실만.
- 자기 PR 금지 — "잘했다" / "수고하셨다" 같은 표현 없음.
