# 삼천리 SL&C — 점포 수익성 & 출점 검토 리뷰 템플릿

> 외식 사업부 월간/분기 임원 보고용. "매출이 아니라 **진짜 남는 매장**"을 브랜드·채널·점포 단위로 가르고, 신규 출점 후보 상권을 데이터로 평가한다.
> 한 줄 호출: `@templates/samchully-slnc/store-profitability-review.md 관점으로`

---

## 이 템플릿이 강제하는 것

- 숫자만 있는 줄은 **미완성**으로 본다. 모든 핵심 숫자 옆에 "그래서 무슨 의미" 한 줄을 붙인다.
- 매출만 보지 않는다. 반드시 **store_master(임대료·변동비·BEP)와 교차**하여 식자재·채널수수료·임대 고정비를 뺀 **영업이익(추정)** 까지 낸다.
- TOP만 나열하지 않는다. **왜 위로 왔는지(공통점)** + **BEP 미달·적자 위험 매장**을 함께 짚는다.
- 출점 검토는 후보지 점수만 나열하지 않고 **기존 매장 상권과 대조**하여 "어느 후보가 어느 브랜드에 맞는지"까지 매핑한다.
- 임원 보고용 코멘트를 **미리 3종(숫자형·흐름형·헤드라인형)** 뽑아 본부장이 고르게 한다.

## 사용 데이터 (반드시 교차)

| 파일 | 쓰임 |
|------|------|
| `pos_sales.csv` | POS 일매출 (매장·브랜드·채널·메뉴카테고리·객수·매출·수수료·식자재비) — 매출의 원천 |
| `store_master.csv` | 매장 임대료·변동비율·BEP 일객수·좌석 — **고정비·손익분기 교차** |
| `menu_master.csv` | 메뉴 카테고리 판매가·원가율 — 메뉴 믹스 분석 |
| `promotions.csv` | 신메뉴·행사 기간 — 매출 급등의 **원인 추정** |
| `trade_area.csv` | 상권 유동인구·오피스인구·임대지수·경쟁밀도·접근성 — **출점 검토** |

**영업이익(추정) 계산식 (매장 단위, 기간 합산):**
```
매출총액      = Σ gross_sales_krw
채널수수료     = Σ channel_fee_krw            (백화점·쇼핑몰만 발생)
식자재비      = Σ food_cost_krw
임대고정비(기간) = monthly_rent_manwon × 만원 × 개월수
영업이익(추정) = 매출총액 − 채널수수료 − 식자재비 − 임대고정비
영업이익률    = 영업이익 ÷ 매출총액
```
> 인건비 등 일부 고정비는 BEP 일객수에 이미 반영된 것으로 보고, 영업이익은 "추정"으로 표기한다.

## 출력 구조

### 1. 한눈 요약 (KPI 카드 4장)

- 총 매출 / 총 영업이익(추정) / 평균 영업이익률 / BEP 미달 매장 수
- **각 카드 아래 한 줄 해석 강제** — 예: "매출은 백화점이 끌지만 수수료 14%가 이익을 깎음 → 오피스가 진짜 남는 채널 (추정)"

### 2. 브랜드별 수익성 (표 + 해석)

표 컬럼: 브랜드 / 매장수 / 매출 / **영업이익(추정)** / **영업이익률** / 객단가 / tier

- **매출 1위 브랜드 ≠ 영업이익 1위 브랜드**인지 명시. 어긋나면 그 자체가 헤드라인.
- 이익률 상위 브랜드의 **공통점**(오피스 비중 높은지, 객단가·회전 구조인지)을 2~3줄.
- **이익률 하위 위험군**: 매출은 크나 이익률 평균 미만인 브랜드 → "볼륨만 큰 저효율" 경고.

### 3. 채널별 수익성 (표 + 해석)

표 컬럼: 채널 / 매장수 / 매출 / 채널수수료 / **영업이익(추정)** / **영업이익률** / 매출순위 / 이익순위

- **매출순위와 이익순위가 어긋나는 채널을 굵게**(table-emphasis-row) 표시하고, 어긋난 이유를 3~4줄로:
  - 백화점·쇼핑몰은 매출은 크지만 입점 수수료(약 14%) + 높은 임대지수가 이익을 깎는다
  - 오피스 상권은 평일 점심 회전이 받쳐 매출 대비 남는다
  - 로드샵은 임대료는 낮으나 평일 집객이 약한지 데이터로 판정

### 4. 점포별 수익성 + BEP 진단 (표 + 경보)

표 컬럼: 매장 / 브랜드 / 채널 / 일평균객수 / **BEP 일객수** / 달성률 / 영업이익(추정) / 상태

- **BEP 미달 매장**(일평균객수 < BEP)을 `alert-chip`으로 경보. 적자 추정 매장은 헤드라인.
- BEP 초과율 상위 매장의 공통점(상권·브랜드·채널)을 1~2줄.
- 매장별로 "왜 부진한가" 추정 1줄 (상권 약세·경쟁 과밀·임대 과중 등, "(추정)" 표기).

### 5. 신규 출점 후보 평가 (표 + 매핑)

표 컬럼: 후보상권 / 유동인구 / 오피스인구 / 임대지수 / 경쟁밀도 / 접근성 / **종합점수** / 추천 브랜드

- 출점 적합도 종합점수 산출(아래 검출 규칙) 후 **상위 후보를 positive-chip**으로.
- 기존 매장 상권의 성과(섹션 4)와 대조: "마곡은 판교형 오피스인데 경쟁밀도가 절반 → Chai797/바른고기 적합 (추정)"
- 각 후보에 **어느 브랜드를 넣어야 하는지** 1줄씩 매핑 (오피스형/쇼핑몰형/프리미엄 주거형 구분).

### 6. 임원 보고용 코멘트 (3종 중 택1 — 미리 제시)

- **숫자형**: "Q1 매출 N억, 영업이익(추정) N억(이익률 N%). 매출 1위 ○○, 이익 1위 △△. BEP 미달 N개점."
- **흐름형**: "매출은 백화점이 끌지만 이익은 오피스가 만든다. 출점은 경쟁 희박한 신흥 오피스로."
- **헤드라인형**: "잘 팔리는 매장과 남는 매장은 다르다 — 다음 출점은 ○○ 상권의 △△."

### 7. 액션 (4열 표)

| 담당 | 액션 | 기한 | 우선순위 | 성공 기준 |
|------|------|------|----------|-----------|
| (예) 점포개발 | 마곡 ○○ 브랜드 출점 타당성 정밀검토 | 차분기 | 상 | 예상 영업이익률 ≥ 평균 |
| (예) 운영 | BEP 미달 ○○점 메뉴믹스·인력 재편 | 차월 | 상 | 달성률 100% 회복 |

## 검출 규칙 (수치 기준)

- **BEP 미달 경보**: 매장 일평균객수 < store_master.bep_daily_guests → 위험군 표기. 달성률 < 80%면 적자 추정 헤드라인.
- **마진 누수 경보**: 브랜드/채널 영업이익률 < 전체 평균 − 5%p → 위험군.
- **저효율 볼륨 경보**: 매출 상위 30% 이면서 이익률 하위 30% → 굵게 + "볼륨만 큰" 표기.
- **매출↔이익 순위 역전**: 매출순위와 이익순위 차이 ≥ 2계단 → 헤드라인 후보.
- **프로모션 효과 검출**: promotions 기간 해당 브랜드 일매출이 평소 평균 대비 +30%↑면 "행사 효과 유효"로 판정.
- **출점 종합점수**: `(유동인구 + 오피스인구) ÷ 임대지수 ÷ 경쟁밀도 × 접근성` 정규화 → 상위 = 1순위 후보. 경쟁밀도 낮고 임대지수 낮은 신흥 상권이 유리하게 잡히도록.

## 길이 / 톤

- A4 2~3장 내. **단, 표 옆 해석/원인 줄은 절대 생략 금지** (표만 있으면 미완성).
- 톤: 영업지원 실무 → 본부장·사업부장 임원 보고. 숫자 옆에 항상 의미. 추정은 "(추정)" 표기.
- 임원 보고용이므로 "그래서 어디에 출점하고, 어느 매장을 손봐야 하는가"의 **의사결정 후보**까지 도달한다.
